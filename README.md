# 🎓 GPA Calculator with Tkinter GUI

A modern desktop application built using Python and Tkinter that helps students calculate both **Semester GPA** and **Cumulative GPA** بسهولة ودقة من خلال واجهة رسومية احترافية وسهلة الاستخدام.

---

# 📌 Project Overview

Many students struggle with:

* Manual GPA calculations
* Calculation mistakes
* Tracking multiple courses
* Understanding GPA status

This project solves these problems through an interactive GUI-based GPA management system.

---

# ✨ Features

## ✅ GPA Calculations

* Semester GPA
* Cumulative GPA
* Weighted GPA Formula

## ✅ Course Management

* Add Courses
* Delete Courses
* Prevent Duplicate Courses
* Maximum 7 Courses per Semester

## ✅ Grade System

* Percentage → Letter Grade Conversion
* GPA Points Mapping
* Academic Status Detection

## ✅ User Experience

* Modern Tkinter GUI
* Real-time Updates
* Error Handling & Validation
* Full-Screen Responsive Interface

# 🏗 System Architecture

The project is divided into 3 main classes:

## 📦 Course

Handles:

* Course Name
* Credits
* Grade Percentage
* Letter Grade
* GPA Points

## ⚙ AcademicSystem

Handles:

* GPA Calculations
* Course Storage
* Semester Management
* Academic Status

## 🖥 GPAApp

Handles:

* GUI Interface
* User Inputs
* Treeview Table
* Dynamic Updates

---

# 📊 GPA Formula

```python
GPA = Σ (Grade Points × Credit Hours) / Σ Credit Hours
```
